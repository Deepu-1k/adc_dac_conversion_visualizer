# ADC/DAC Conversion Visualizer (MVP)

This project is a simple interactive MVP that demonstrates:
- Analog signal visualization
- Analog-to-Digital Conversion (ADC)
- Sampling rate effect
- Bit resolution effect
- Conceptual Digital-to-Analog Conversion (DAC)

## Technologies Used
- HTML
- CSS
- JavaScript
- Canvas API
